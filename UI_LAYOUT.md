## Responsive Features

1. **Scrollable Lists**: Both contact list and edit form have vertical scrollbars when content exceeds viewport
2. **Window Resize**: Content scales proportionally
3. **Touch-Friendly**: All elements use 16px+ fonts and have generous padding
4. **Spacing**: 10-20px margins between elements for easy clicking
